アップデートノート 更新说明 v2.0.0

1.5周年のアップデートノートへようこそ！
こんにちは、『プロジェクトセカイ』開発チームです。

久しぶりのメジャーアップデートとなるバージョン2.0.0では、主に新しい遊び方となる「ランクマッチ」の実装が行われます。

欢迎阅读1.5周年更新日志！

大家好，这里是『Project SEKAI』开发团队。

在本次时隔已久的大型更新2.0.0当中，主要将实装作为全新的游玩方式的「排位赛」。

▶「排位赛」的实装与预热赛季的举行

リリースからこれまでの間、『プロジェクトセカイ』では定期的に「プロセカCS」等の公式大会を開催してまいりました。
私達としては、プロセカにおけるリズムゲームパートと競技シーンの紐付けは今後も強化していきたい点のひとつだと考えており、この度、より公式大会に近い環境で腕前を競うことができるゲームモードを追加することにしました。

  • ランクマッチについて
    「ランクマッチ」は、シーズンと呼ばれる期間内に上位のランクを目指し、プレイヤー同士が1vs1で腕前を競う対戦式のリズムゲームモードです。
    ランクマッチはイベントとしてではなく一つのゲームモードとして追加されるため、他のイベントの開催とは関係なく、並行して遊ぶことができます。

  • ランクマッチへの参加方法
    ランクマッチはリズムゲームの腕前を競う少し難易度の高いモードとなるため、以下の条件をクリアすることで初めて出現し、参加可能となります。

  • ランクマッチ解放条件
    楽曲難易度「EXPERT」または「MASTER」のフルコンボ合計数が10以上

自从游戏上线至今，『Project SEKAI』已经定期举行了诸如「世界计划CS」等的官方大赛。

对我们来说，『世界计划』的节奏游戏部分和竞技场景的联系是今后也想要继续强化的一点，因此，本次决定追加能够在更加接近官方大赛的环境中比拼技术的游戏模式。

  • 关于排位赛

    「排位赛」是指在被称为赛季的时间段内向高段位发起冲击，玩家之间以1v1形式比拼技术的对战型节奏游戏模式。

    排位赛不是作为活动，而是作为一种游戏模式追加，所以与其他活动无关，可以并行游玩。

    • 参加排位赛的方法

      排位赛是是比拼玩家的节奏游戏技术的难度较高的模式，所以需满足以下条件才会首次出现，并进行游玩。

    • 排位赛的开放条件

      乐曲难度「EXPERT」或「MASTER」的FULL COMBO总数达到10以上

  • ランクマッチのルールと楽曲難易度について
    ランクマッチは、以下の2つの条件に基づく特別ルールで実施します。

    • 編成ユニット

      • 編成ユニットの総合力はスコアに影響しません。
      • 編成メンバーのスキルは発動しません。

    • スコア計算
      以下のノーツ判定結果によってスコアが計算されます。

      • PERFECT：＋3
      • GREAT　 ：＋2
      • GOOD　 ：＋1
      • BAD　　 ：±0
      • MISS　 ：±0

※スコアが同点の場合、①PERFECT数、②最大コンボ数、③残りライフの優先度で勝敗を決定し、これらの項目でも勝敗がつかない場合は引き分けとなります。

なお、ランクマッチでもライブ報酬・楽曲報酬・各種EXP・イベントPやバッジなどの報酬を獲得することができます。
こちらの報酬を決めるスコアは、デッキに編成されている総合力とライブ結果を元とした、通常通りのスコア計算によって行われ、また、獲得できる報酬の量は「ひとりでライブ」と同量になり、ライブボーナスも適用されます。

  • 対戦で選曲される楽曲について
    対戦では自身のランクに応じて、ランダムで楽曲が選曲されます。
    各ランク帯で選曲される楽曲の難易度レベルは以下の通りです。

    ビギナーランク　　：22 ～ 24
    ブロンズランク　　：23 ～ 26
    シルバーランク　　：25 ～ 28
    ゴールドランク　　：27 ～ 30
    プラチナランク　　：29 ～ 32
    ダイヤモンドランク：30 ～ 34
    マスターランク　　：31 ～ 36

※上記の難易度レベルについては、本アップデートノート下部に記載しております「難易度レベル幅拡張と一部楽曲の難易度レベル見直し」適用後のものとなります。詳細は該当項目をご一読ください。

  • 关于排位赛的规则和曲目难度

    排位赛将会实施基于以下两个条件的特殊规则。

    • 编队

      • 编队的综合力对分数没有影响。

      • 编队成员的技能不会发动。

    • 计分

      ​ 根据按键的判定结果按照以下规则计算得分。

      • PERFECT：+3
      • GREAT：+2
      • GOOD：+1
      • BAD：±0
      • MISS：±0

※分数相同的情况下，按照①PERFECT数；②最大combo数；③剩余生命值的优先级决定胜负，若根据上述项目仍然不能决定胜负，则视为平局。

另外，在排位赛中也可以获得演出报酬·乐曲报酬·各种经验值·活动点数和徽章等报酬。

决定这些报酬的分数，是以原编队的综合力和演出结果为基础，根据通常的计分法计算的，另外，能够获得的报酬数量和「单人演出」相同，并适用演出加成。

  • 关于在对战中选择的曲目

    在对战中将根据玩家自身的段位，随机选择曲目。

    各段位中会被选择的曲目难度等级如下所示。

    新手段位（beginner rank）：22~24

    青铜段位（bronze rank）：23~26

    白银段位（silver rank）：25~28

    黄金段位（gold rank）：27~30

    白金段位（platina rank）：29~32

    钻石段位（diamond rank）：30~34

    大师段位（master rank）：31~36

※上述的难度等级，是指本更新日志后文记载的「难度等级范围扩张以及部分曲目的难度等级修改」适用后的难度等级。详见该对应项目。

  • ランクマッチのランクとシーズンについて

    • スタートランクについて
      ランクマッチではスタートするランクを決定するため、現在の実力を計る対戦を3回行います。この対戦では、過去のライブのプレイ実績を元に、近い実力同士のプレイヤーとのマッチングとなります。この3戦の結果と過去のライブのプレイ実績を元にスタートランクが決定します。

    • ランクについて
      ランクを昇格させるには「ランクポイント」をためる必要があります。
      ライブに勝利することで1つ獲得でき、このランクポイントを5つためた状態で対戦に勝利することで上のランクに昇格できます。
      逆に、ライブに敗北するとランクポイントは1つ減ってしまいます。ランクポイントが0の状態で敗北すると下のランクに降格となります。

    • ランクポイントボーナスについて
      ランクポイントには、100ためることでランクポイントを1つ増やすことができる「ランクポイントボーナス」があります。
      ランクポイントボーナスはライブ結果によって増減します。プレシーズン開催時、条件は以下の通りですが、今後調整が加わる可能性がございます。

      ・ALL PERFECT　：＋20
      ・フルコンボ 　　 ：＋10
      ・上位クラスに勝利：+10
      ・ライフ0　　　　 ：-30
      ※「ALL PERFECT」と「フルコンボ」のポイントボーナスは重複しません。

    • シーズン
      ランクマッチは、一定の期間を1シーズンとし、その期間内にどれだけ上位のランクを目指せるかを競うシーズン制となります。
      シーズン終了時点でのランクが最終戦績となり、そのシーズンランクを報酬である「称号」として獲得することができます。

次のシーズンが始まるとランクはリセットされ、スタートランクから始めることになります。
なお、過去のシーズンでの最終ランクは記録されるため、ランクマッチ画面からいつでも確認することができます。

  • 关于排位赛的段位和赛季

    • 关于初始段位

    为了决定玩家在排位赛中的初始段位，将进行3次测试当前实力的对战。该对战将以玩家过去的演出成绩为基准，匹配实力相近的玩家。根据这3次对战的结果和过去的演出成绩，决定初始段位。

    • 关于段位

    要提升段位需要累积「段位点数」。

    每胜利一场可以获得1个段位点数，在已经累积5个段位点数的状态下再次获胜即可提升至上一段位。

    相反，每失败一场会减少1个段位点数。段位点数为0的状态下失败则降至下一段位。

    • 关于段位点数奖励积分

    对于段位点数，存在每累积100分即可增加1个段位点数的「段位点数奖励积分」。

    段位点数奖励积分会根据演出结果增减。预热赛季开始时，条件暂时如下，在今后可能会进行调整。

    •ALL PERFECT：+20

    •FULL COMBO：+10

    •战胜分段更高的对手：+10

    •生命值为0：-30

    ※「ALL PERFECT」和「FULL COMBO」的奖励积分不会叠加。

    • 赛季

    排位赛以一定的时间段为一个赛季，实行在该时间段内尽可能争取高段位的赛季制。

    赛季结束时的段位为最终战绩，可以获得以赛季段位为「称号」的报酬。

    下一赛季开始时段位将会重置，从初始段位开始比赛。

    另外，在过去的赛季中的最终段位会被保存下来，随时可以在排位赛界面确认。

  • プレシーズンの開催について
    ランクマッチは実際に本シーズンを開始する前に、プレシーズンを実施いたします。

    プレシーズンでは、ランク内のプレイヤー数が過度に偏ることがないなどのバランス調整の他、サーバへの負荷、各機能において深刻な問題が発生しないかなど、本シーズンを適切な状態で開催するためのデータ収集と確認を行います。

    プレシーズンの開催にあたり、ランクマッチの参加人数の計測、及び全ランク帯で安定したマッチングを成立させるため、ランクマッチに参加できる時間を18:00～24:59の限定開催とさせていただきます。
    参加人数やマッチングなどが安定した状態での開催が確認できれば、プレシーズン期間内に開催時間を拡大することも検討しております。

    また、プレシーズンは本シーズンと異なり、シーズン終了時の報酬はごく少量のコインのみとなり、戦績も履歴には残りません。
    その分、本シーズンに向けたウォームアップやランクマッチ機能の体験として、気兼ねなく触れていただければと思います。

    プレシーズンの開催時期などは別途ゲーム内のお知らせでご案内する予定ですので、楽しみにお待ちください。

ユニットランクが大幅に上がりやすくなるよう、ライブで獲得できるユニットEXP量の増加

一方、ライブを遊んでメインストーリーを解放するといった一連のゲームプレイの流れは、ゲームの遊び方に慣れていただくための重要な立ち位置とも考えており、この度、ライブで獲得できるユニットEXPを大幅に増加する調整を行うことにいたしました。

この調整は、1.5周年のメインストーリー解放キャンペーン終了後から適用する予定です。

一部ガチャの廃止、開催期間の調整

リリース時より様々な種類のガチャをお届けしてまいりましたが、始めたばかりの方々にとって『結果的にどれを引けばよいのか』が分かりづらい状態となっておりました。
これを解消するため、今後一部のガチャを廃止、また開催期間の調整をいたします。

対象となるガチャと詳細については以下の通りです。

○イベントメンバーガチャ
リリース初期は、イベントボーナスを得られるメンバーが1人もいない方々が多かったため、開催されるようになったガチャですが、現在、その役目は終わりつつあります。3月30日以降に開催予定のイベントより廃止いたします。

○★3以上確定チケットガチャ
貴重な「想いのカケラ」の使い先としては、あまり挙がらない候補となってしまっております。そのため、4月30日14時59分をもって廃止いたします。
※本ガチャの廃止に伴い「★3以上確定ガチャチケット」も廃止となります。所持されている「★3以上確定ガチャチケット」は後日回収し、1個あたり「想いのカケラ×300」の返還を行う予定です。
※本ガチャの廃止に伴い、毎月月中で更新していた「★3以上確定チケットガチャ」の4月の更新は実施いたしません。

○初心者応援スタートダッシュガチャ
「初心者応援メモリアルガチャ」とどちらを引けば良いか、わかりにくい状態になってしまっておりました。そのため4月30日14時59分をもって廃止いたします。代わりに「初心者応援メモリアルガチャ」の開催期間をスタートダッシュガチャと同様の1週間に延長します。

○初心者応援メモリアルセレクトガチャ
4月28日13時30分頃より開催期間をチュートリアル突破後2日間から1週間に変更いたします。
※上記の適用開始時間は予告なく変更となる場合がございます。

■关于预热赛季的举行

排位赛在正式赛季开始之前，会实施预热赛季。

在预热赛季中，在调整同一段位内的玩家数过多或过少等影响平衡的现象的同时，还将收集和确认诸如服务器负荷、各功能是否有严重问题等，确保正式赛季能够在正常状态下举行的数据。

为了保证在预热赛季举行期间，能够测算参加排位赛的人数，并保证所有段位内的匹配都能安全进行，预热赛季内排位赛的参加时间限于18:0024:59（北京时间17:0023:59）之间。如果能够确定参加人数和匹配等均处于稳定状态，也会考虑在预热赛季内放宽参加时间。

另外，预热赛季与正式赛季不同，赛季结束时的报酬仅为极少量的金币，战绩和记录也不会保留。

因此，也希望各位玩家能把预热赛季作为正式赛季的热身运动以及对排位赛功能的体验，放下顾虑大胆尝试。

预热赛季的举行时间等信息将在游戏内通知中另行告知，敬请期待。

▶★4成员和生日（纪念日）成员的master rank加成数值上升

在1周年时实施了「与master rank对应的活动点数和徽章加成」，但是与提升★4成员和生日（纪念日）成员的master rank付出的劳动相比，回报太少，所以将按照下列规则提升加成数值。

★4成员

・master rank 1： 2% → 10%

・master rank 2： 4% → 11%

・master rank 3： 6% → 12%

・master rank 4： 8% → 13%

・master rank 5：10% → 15%

生日（纪念日）成员

・master rank 1：1.5% → 5%

・master rank 2： 3% → 6%

・master rank 3：4.5% → 7%

・master rank 4： 6% → 8%

・master rank 5：7.5% → 10%

▶亲密等级的适用组合追加

在亲密等级中将追加以下3个组合。

•天马咲希&天马司

•日野森志步&日野森雫

•东云彰人&东云绘名

另外，即使在亲密等级开放前使用指定的组合游玩，亲密等级经验值也不会事先积累。还请谅解。

本次追加的仅有上述3组，但是，也有在进行能在今后加快节奏追加更多组合的准备。现在还请稍作等候。

▶难度等级范围扩张以及部分曲目的难度等级修改

随着被称为高难度乐曲的曲目增加，一直以来的难度等级表示的范围，特别是33~34难度的范围里，即使是同等级的曲目，每首乐曲之间的难度体感也大有不同。

▶为了让组合等级更容易上升，增加在演出中获得的组合经验值

现在，对于刚开始游玩『世界计划』的人，解锁主线剧情所需的组合经验值太难获取，从而导致阅读进度无法推进的状态一直在持续着。

另外，我们认为，通过演出解锁主线剧情的一套流程是熟悉本游戏游玩方式的重要一环，因此，本次调整将大幅提升演出中能够获得的组合经验值。

此调整预定自1.5周年主线剧情开放活动结束后开始生效。

▶部分卡池废止、举办期间调整

自从游戏上线开始，我们设置了种类众多的卡池，结果对刚开始游玩的人来说，造成了难以分辨『结果到底该抽哪个好』的结果。

为了消除这种状态，今后将废止一部分卡池，或调整部分卡池的举行期间。

将进行调整的卡池内容如下所述。

○活动成员招募

游戏上线初期，由于有不少玩家连一个能获得活动加成的成员都没有，设置了本卡池，现在，本卡池的任务已经完成。预定3月30日以后开始的活动中将废止本卡池。

○★3以上确定招募

在贵重的「心愿碎片」的使用上，本卡池成为了基本不会被选择的选项。因此，从4月30日14时59分（北京时间4月30日13时59分）开始本卡池将废止。

※与本卡池的废止相对应，「★3以上确定抽卡券」也将废止。玩家目前持有的「★3以上确定抽卡券」将在日后被回收，每回收1张相应返还「心愿碎片×300」。

○新手应援冲刺卡池

本卡池与「新手应援纪念卡池」，让玩家难以判断抽哪个更好。因此从4月30日14时59分（北京时间4月30日13时59分）开始将废止本卡池。作为代替，「新手纪念卡池」的举行期间将延长至与本卡池相同的一周。

○新手纪念自选卡池

从4月28日13时30分（北京时间4月28日12时30分）左右开始，本卡池的举行期间将从新手教程完成后2天内变更为1周内。

※以上记载的适用开始时间有可能不经预告而发生改变。
